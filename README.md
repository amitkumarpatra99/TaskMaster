## ✨ Features

- **Glassmorphic UI**: Beautiful frosted-glass container with subtle backdrop blurs and fluid shadows.
- **Dynamic Animations**: Smoothly animated blob backgrounds and keyframe transitions on interactions.
- **Local Storage API**: Your tasks are saved locally within your browser, ensuring nothing is lost upon refreshing!
- **State Filtering**: Quickly isolate tasks by switching between *All*, *Active*, and *Completed* tabs.
- **Micro-Interactions**: Hover states, custom animated checkboxes, and deletion effects make managing tasks feel incredible.

## 🛠️ Technology Stack

- **[React 18](https://react.dev/)**: Reactive UI state management
- **[Vite](https://vitejs.dev/)**: Lightning-fast Hot Module Replacement (HMR) and optimized build tool
- **Vanilla CSS3**: Highly customized variables, floating animations, and responsive flexbox layouts
- **[Lucide React](https://lucide.dev/)**: Clean, robust SVG iconography

## 🚀 Getting Started

Follow these steps to get your development environment running locally.

### Prerequisites

- [Node.js](https://nodejs.org/) installed (v16.0 or higher recommended).
- `npm` (comes with Node.js).

### Installation Setup

1. **Clone or Open the Repository**
   Make sure your terminal is inside the `AMIT PATRA` directory.
   ```bash
   cd "AMIT PATRA"
   ```

2. **Install Dependencies**
   Install the necessary node modules for the project.
   ```bash
   npm install
   ```
   *(Note: if PowerShell restricts script execution, use `npm.cmd install`)*

3. **Run the Development Server**
   Spin up Vite's blazingly fast development environment.
   ```bash
   npm run dev
   ```
   *(Note: if PowerShell restricts script execution, use `npm.cmd run dev`)*

4. **Start Managing Tasks!**
   Open your browser and navigate to `http://localhost:5173`.

## 🎨 Design Philosophy

This application focuses heavily on *user sentiment*. An ordinary Todo App simply lists text. We aim to inject a premium "wow" factor into mundane tasks through a curated dark color palette, glowing gradients, and zero-latency visual feedback on every action.
